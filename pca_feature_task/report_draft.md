# DNN実践課題2 第2題: PCAによるDCNN特徴の次元削減

## 1. 目的

主成分分析（Principal Component Analysis, PCA）を用いて、DCNNから得られる高次元画像特徴を低次元に圧縮し、クラスタリング結果がどのように変化するかを確認する。PCAは、データの分散が大きい方向を主成分として取り出す線形次元削減手法である。

## 2. 方法

データセットにはCIFAR-10を用いた。画像をImageNetの入力形式に合わせて224x224 pixelへリサイズし、ImageNetで学習済みのVGG16に入力した。特徴量として、VGG16のfc7層から得られる4096次元特徴を用いた。

PCAは500枚の画像特徴で学習し、そのうち100枚を用いてクラスタリング結果を比較した。比較した特徴表現は、元の4096次元特徴、PCAで累積寄与率95%となる次元、累積寄与率90%となる次元、128次元に圧縮した特徴である。それぞれについて、k-meansをk=5およびk=10で実行した。

## 3. 結果

PCA後の次元数を以下に示す。

![pca dimensions](figures/pca_dimensions.png)

元のDCNN特徴は4096次元である。PCA後の次元数は、累積寄与率95%では304次元、累積寄与率90%では209次元となった。また、128次元に固定した場合の累積寄与率は0.828331であり、元の4096次元特徴の約82.8%の分散を保持していた。

クラスタリング結果の例を以下に示す。

![4096 k5](figures/clusters_4096dim_k5.png)

![pca95 k5](figures/clusters_pca95_k5.png)

![pca90 k5](figures/clusters_pca90_k5.png)

![pca128 k5](figures/clusters_pca128_k5.png)

k=5の結果を見ると、4096次元特徴とPCA 95%特徴では非常に近いクラスタ構成が得られた。例えば、4096次元のcluster 4では automobile:16 / truck:12 が多く、PCA 95%でも automobile:16 / truck:12 が同じクラスタに集まった。また、bird, horse, deer が多いクラスタも4096次元とPCA 95%でほぼ一致した。

## 4. 考察

4096次元特徴はDCNNがImageNetで学習した視覚的特徴を含んでいるため、CIFAR-10に対しても見た目が似た画像が近くなりやすい。PCAで4096次元から304次元まで圧縮しても、クラスタリング結果が大きく変化しなかったことから、主要な分散情報はPCA後にも保持されていたと考えられる。

一方で、128次元では累積寄与率が約82.8%まで下がるため、4096次元やPCA 95%と比べると細かいカテゴリ差を表す情報は失われやすい。それでも重要な主成分を優先して残しているため、完全にランダムなクラスタリングにはならず、見た目が近い画像がある程度まとまった。

## 5. まとめ

VGG16の4096次元特徴をPCAで次元削減し、k-meansクラスタリングを比較した。累積寄与率95%では304次元、90%では209次元まで圧縮できた。特にPCA 95%では4096次元特徴と近いクラスタリング結果が得られ、高次元特徴を大きく圧縮しても主要な画像特徴を保持できることが確認できた。
