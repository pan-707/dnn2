# DNN実践課題2 第2題: PCAによるDCNN特徴の次元削減

## 1. 目的

主成分分析（Principal Component Analysis, PCA）を用いて、DCNNから得られる高次元画像特徴を低次元に圧縮し、クラスタリング結果がどの程度変化するかを調べた。PCAは、データの分散が大きい方向を主成分として取り出す線形の次元削減手法である。

## 2. 方法

データセットにはCIFAR-10を用いた。画像をImageNetの入力形式に合わせて224x224 pixelへリサイズし、ImageNetで学習済みのVGG16に入力した。特徴量には、VGG16のfc7層から得られる4096次元特徴を用いた。

PCAは500枚の画像特徴で学習し、そのうち100枚を使ってクラスタリング結果を比較した。比較した特徴表現は、元の4096次元特徴、累積寄与率95%のPCA特徴、累積寄与率90%のPCA特徴、128次元に固定したPCA特徴である。それぞれについて、k-meansをk=5およびk=10で実行した。

## 3. 結果

PCA後の次元数を以下に示す。

![pca dimensions](figures/pca_dimensions.png)

元のDCNN特徴は4096次元である。PCA後は、累積寄与率95%で304次元、90%で209次元となった。128次元に固定した場合の累積寄与率は0.828331であり、元の特徴の分散を約82.8%保持していた。

クラスタリング結果の例を以下に示す。

![4096 k5](figures/clusters_4096dim_k5.png)

![pca95 k5](figures/clusters_pca95_k5.png)

![pca90 k5](figures/clusters_pca90_k5.png)

![pca128 k5](figures/clusters_pca128_k5.png)

k=5の結果では、4096次元特徴とPCA 95%特徴でかなり近いクラスタ構成になった。例えば、4096次元のcluster 4では automobile:16 / truck:12 が多く、PCA 95%でも automobile:16 / truck:12 が同じクラスタに集まった。また、bird, horse, deer が多いクラスタも、4096次元とPCA 95%でほぼ同じだった。

## 4. 考察

VGG16の4096次元特徴には、ImageNetで学習された視覚的な情報が含まれている。そのため、CIFAR-10に対しても、見た目の近い画像が近い特徴として表現されやすい。PCAで304次元まで圧縮してもクラスタリング結果が大きく崩れなかったことから、主要な情報はかなり残っていたと考えられる。

一方、128次元では累積寄与率が約82.8%まで下がる。4096次元やPCA 95%と比べると、細かいカテゴリ差を表す情報は失われやすい。それでも、完全にばらばらなクラスタにはならず、見た目が近い画像はある程度まとまっていた。

## 5. まとめ

VGG16の4096次元特徴をPCAで次元削減し、k-meansクラスタリングを比較した。累積寄与率95%では304次元、90%では209次元まで圧縮できた。特にPCA 95%では4096次元特徴に近いクラスタリング結果が得られ、高次元特徴を大きく圧縮しても主要な画像特徴は残ることが分かった。
